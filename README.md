# 🚀 NEXA AI — Multi-Model AI Chatbot

NEXA AI is a **next-generation multi-model AI assistant** that allows users to interact with multiple AI models like **ChatGPT, Gemini, and Claude** in a single interface.

---

## 🌟 Features

✨ Multi-AI Support

* ChatGPT (OpenAI)
* Gemini (Google AI)
* Claude (Anthropic)

⚡ Dual Mode

* Single AI Mode
* Multi-AI Comparison Mode

🔑 API Key Management

* Store API keys securely in localStorage
* Toggle visibility

💬 Smart Chat UI

* Real-time messaging
* Typing animation
* Chat history

🎨 Modern UI/UX

* Dark theme
* Glassmorphism design
* Smooth animations

🎤 Advanced Features

* Voice input (Speech Recognition)
* Text-to-Speech (TTS)
* Copy responses
* Regenerate answers

## ⚙️ Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/your-username/NEXA-AI.git
cd NEXA-AI
```

### 2. Run Project

Just open:

```bash
index.html
```

(No backend required)

---

## 🔑 API Setup

You need API keys for:

* OpenAI → https://platform.openai.com/
* Gemini → https://makersuite.google.com/
* Claude → https://console.anthropic.com/

Add them in the sidebar inside the app.
